Implementing the Iterable JavaScript SDK

In this project, I examine every shot that Kobe Bryant took during his career in order to build his shot profile. The eventual goal of this project is to train a Machine Learning Model to predict whether Kobe would make or miss a shot based on sample data.

Create Iterable User Profile

This is an demo of how an iterable user profile can be generated. In this example, user identifiers such as email, firstName, isWebUser, and SA_WebUser_Test_Key will be passed to iterable via an API call and stored in the database.

Create a Custom Event

This is an example of the trackEvent method in Iterable's JS SDK. This method is used to send custom events along with custom payloads.
